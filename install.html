🕒 ExcelSheet Time Filler v0.1.0

Remplit automatiquement la grille de saisie des temps de Project Web App (PWA) à partir d'un template de semaine que tu définis une seule fois.

1. Installer le bookmarklet

Glisse le bouton ci-dessous dans ta barre de favoris (affiche-la avec Ctrl+Maj+B si besoin) :

📌 ExcelSheet Time Filler
Glisser-déposer ce bouton dans la barre de favoris de ton navigateur

2. Utiliser l'outil

  1. Va sur ta page Tâches dans ExcelSheet
  2. Clique sur le favori ExcelSheet Time Filler installé à l'étape 1
  3. Un panneau flottant apparaît : renseigne tes lignes (tâche + valeur par jour)
  4. Clique sur « 💾 Enregistrer le template » pour le sauvegarder localement
  5. Clique sur « ▶ Appliquer sur la semaine » pour lancer le remplissage automatique

Alternative sans installation

Tu peux aussi coller directement le contenu de pwa-time-filler.js dans la console développeur (F12) de la page Tâches ExcelSheet, sans passer par le favori.

Notes

  • Le template est sauvegardé dans le localStorage de ton navigateur : il reste disponible tant que tu ne vides pas les données du site.
  • Les tâches à remplir doivent être visibles dans la grille (pas de groupe replié) au moment de l'exécution.
